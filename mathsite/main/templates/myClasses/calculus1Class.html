Domain, Range and Functions

A function $F$ is a rule that takes a number $x$ and assigns to it a unique (single) number $f(x)$.

Examples: $f(x) = 3x - 2$ and $f(x) = x^2 + 1$

The set of inputs (the number of $x$'s) are called the domain. The set of outputs (the number of output) are called the range.

Example: Find the domain and range of the following equation: $f(x) = x^3 + 1$

We can plug in any $x$ and we will get an output. Therefore, the set of all real numbers, $\mathbb{R}$, is the domain of $f(x)$. This can also be written as $(-\infty, \infty)$. However, it' better to specify that all real numbers are the domain. Later on we will end up seeing some cases where some specific $x$ values will not be allowed.

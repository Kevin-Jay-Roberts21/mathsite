Ordinary Differential Equations: MATH 2280

A differential equation is an equation involving some unknown function and it's derivatives.

$u(x) = \frac{du}{dx}$

Independent and Dependent Variables

Observe the following two equations:

$\frac{dy}{dx} - 4x^2 = 0$ and $y''' - 3y' = t$

For the first equation, $x$ is the independent variable and $y$ is the dependent variable. For the second equation, $t$ is the independent variable and $y$ is a dependent variable. We say they are "dependent" if they "depend" on something. For example, the change of $y$ depends on the change in $x$. $x$ is what's controlling the equation and the outcome, where the outcome, $y$, depends on what the independent variable $x$ is.

Try spotting the independent and dependent variables in the following equations:

$\frac{dp}{dt} = -10p$  $\frac{dp}{dt} = 0.3tp - 0.006p^3$  $\frac{\partial u}{\partial t} - 6\frac{\partial u}{\partial x} = 0$

ODE's and PDE's

A differential equation involving ordinary derivatives is called an ordinary differential equation or (ODE) and a differential equation that involves partial derivatives is called a partial differential equation or (PDE).

ODE: $x^2y'' - 6xy' + 4y = x$, where $y'' = \frac{d^2y}{dx^2}$ and $y' = \frac{dy}{dx}$

PDE: $\frac{\partial' u}{\partial t^2} - \frac{\partial' u}{\partial x^2} = \sin(x*y)$

Order of Equations

The order of any differential equation is given by the largest order derivative:

2nd Order PDE: $\frac{\partial' u}{\partial t^2} - \frac{\partial' u}{\partial x^2} = \sin(x*y)$

2nd Order ODE: $x^2y'' - 6xy' + 4y = x$

1st Order ODE: $\frac{dp}{dt} = 3p - 0.01p^2$

Linear and Non-linear

An equation that involves non-linear terms in the dependent variable is called a non-linear differential equation. Otherwise the equation is linear:

Linear, 1st Order ODE:   $\frac{dp}{dt} = -4p$

Non-linear, 1st Order ODE: $\frac{dp}{dt} = -4p -6p^2$

Non-linear, 1st Order PDE: $u\frac{\partial u}{\partial x} - \frac{\partial u}{\partial t^2} = 0$

Example: Finding the general solution to a differential equation:

$\frac{d^2y}{dx^2} - 4x^2 = 0$

This is a second order linear differential equation. The first step is to solve for $y$.

$\frac{d^2y}{dx^2} - 4x^2 = 0 \implies \frac{d^2y}{dx^2} = 4x^2$

Now integrate both sides with respect to x:

$\int{\frac{d^2y}{dx^2}}dx = \int{4x^2}dx \implies \frac{dy}{dx} + C_1 = \frac{4}{3}x^3 + C_2 \implies \frac{dy}{dx} = \frac{4}{3}x^3 + C_2 - C_1$

$C_1$ and $C_2$ are just unkown constants. When they are added together, we just get another constant, let's call it $C_3$. Now we have the following:

$\frac{dy}{dx} = \frac{4}{3}x^3 + C_3$ and integrate again:

$\int{\frac{dy}{dx}}dx = \int{\frac{4}{3}x^3 + C_3}dx \implies y + C_4 = \frac{1}{3}x^4 + C_3x + C_5 \implies y = \frac{1}{3}x^4 + C_3x + C_5 + C_4$

Combine constants $C_4$ and $C_5$ to get $C_6$, and our final answer is:

Solution:$y = \frac{1}{3}x^4 + C_3x + C_6$

Recap: We solved the differential equation $\frac{d^2y}{dx^2} - 4x^2 = 0$ for $y$ and ended up getting $y = \frac{1}{3}x^4 + C_3x + C_6$ as a general solution. Thus, we can pick any combination of values of $C_3$ and $C_6$ to get an exact (or unique) solution. Possible unique solutions could be $y = \frac{1}{3}x^4 + 5x + 63$ or $y = \frac{1}{3}x^4 + 2x + 0.001$. In general, to find unique solutions, you must be given intial conditions.


Example: Finding the solution to a differential equation with initial conditions:

$\frac{dy}{dx} = \sin(x)$, with $y(\pi) = 3$

This is a first order linear differential equation. Let's start by integrating both sides with respect to $x$

$\int{\frac{dy}{dx}}dx = \int{\sin(x)}dx \implies y + C_1 = -\cos(x) + C_2 \implies y = -\cos(x) + C_2 - C_1$

Let's combine constants $C_1$ and $C_2$ and define it as $C_3$, then our general soltuion is the following:

$y = -\cos(x) + C_3$

This is the general solution to the intial problem, but we were given intial conditions this time, so let's use them and find the exact solution.

If $y(\pi) = 3$ and $y(\pi) = -\cos(\pi) + C_3$, then $3 = -\cos(\pi) + C_3 \implies C_3 = 3 + \cos(\pi) \implies C_3 = 3 - 1 = 2$

We just solved for $C_3$, so now let's rewrite our final, unqiue solution as:

Solution: $y = -\cos(x) + 2$

Recap: We solved for the general solution of $\frac{dy}{dx} = \sin(x)$ and found it to be $y = -\cos(x) + C_3$. After using intial condition, $y(\pi) = 3$,we found the exact (or unique) solution to be $y = -\cos(x) + 2$.

We can specify conditions on $y$ and it's derivatives to fix the constants of integration. If we specify the conditions at the same point, they're called initial conditions, (or initial values). We just did this in the last example.

Example: Finding the solution to a differential equation with initial conditions:

$\frac{d^2y}{dx^2} = t - e^{-t}$ with $y(1) = -1$, $y'(1) = 1$

This is a second order linear differential equation. Start solving by integration:

$\int{\frac{d^2y}{dt^2}}dt = \int{t - e^{-t}}dt \implies \frac{dy}{dt} = \frac{1}{2}t^2 + e^{-t} + C_1$

For the above, I just cut to the chase and combined the constants into one and called it $C_1$. Now let's integrate again:

$\int{\frac{dy}{dt}}dt = \int{\frac{1}{2}t^2 + e^{-t} + C_1}dt \implies y = \frac{1}{6}t^3 - e^{-t} + C_1t + C_2$

Again, I just cut to the chase and named the next combination of constants $C_2$. We have solved for the general solution. Now let's use the inital conditions and find the exact solution. The goal is to figure out what $C_1$ and $C_2$ are:

Recall the first derivative of our problem is $\frac{dy}{dt} = y'(t) = \frac{1}{2}t^2 + e^{-t} + C_1$. If $y'(1) = 1$, then...

$1 = \frac{1}{2}(1)^2 + e^{-(1)} + C_1 \implies C_1 = 1 - \frac{1}{2} - \frac{1}{e} \implies C_1 = \frac{1}{2} - \frac{1}{e}$

If $y(1) = -1$, then...

$-1 = \frac{1}{6}(1)^3 - e^{-(1)} + C_1(1) + C_2 \implies C_2 = e^{-1} - C_1 - 1 - \frac{1}{6} \implies C_2 = \frac{1}{e} - C_1 - \frac{7}{6}$

We know what $C_1$ is, so let's plug is in to the above equation and solve for $C_2$:

$C_2 = \frac{1}{e} - (\frac{1}{2} - \frac{1}{e}) - \frac{7}{6} \implies C_2 = \frac{1}{e} - \frac{1}{2} + \frac{1}{e} - \frac{7}{6} \implies C_2 = \frac{2}{e} - \frac{5}{3}$

Now let's plug in $C_1$ and $C_2$ into our gernal solution:

$y = \frac{1}{6}t^3 - e^{-t} + (\frac{1}{2} - \frac{1}{e})t + (\frac{2}{e} - \frac{5}{3}) \implies y = \frac{1}{6}t^3 - e^{-t} + \frac{1}{2}t - \frac{1}{e}t + \frac{2}{e} - \frac{5}{3}$

With some simplification, the final solution is the following:

Solution: $y = \frac{1}{6}t^3 - e^{-t} + \frac{e - 2}{2e}t + \frac{6 - 5e}{3e}$

Recap: We solved for the general solution and found it to be $y = \frac{1}{6}t^3 - e^{-t} + C_1t + C_2$. After using intial conditions, we solved for $C_1$ and $C_2$, then plugged them back into our general solution to find an exact solution.


Directly Integrable ODE's

So far we have been lucky enough to have examples that are directly integrable. A directly integrable differential equation is of the following form:

$\frac{dy}{dx} = F(x)$, where $F(x)$ is a function of x.

Here are some more examples of differential equations that are directly integrable:

1st Order: $\frac{dy}{dx} = f(x) \rightarrow$

  Ex: $\frac{dy}{dx} = -e^{-x} \implies y(x) = -e^{-x} + C$

2nd Order: $\frac{d^2y}{dx^2} = g(x) \rightarrow$

  Ex: $\frac{d^2y}{dx^2} + 4\frac{dy}{dx} = h(x) \implies \frac{dy}{dx} + 4y = \int{h(x)}dx + C$

3rd Order: $y''' = x*\cos(x)$

In every example above we have done indefinite integration. This is when we don't place bounds in the integration step. Also I'd like to make a note (we'll be doing this in the next example) that $\frac{dy}{dx}$ is the same as $y'$. Here is another example of solve a differential equation where integration is indefinite:

Example: Finding the solution to a differential equation with initial conditions:

$y' = x + \cos(x)$ with $y(1) = 7$

This is a first order linear differential equation. Start solving by integration:

$\int{y'}dx = \int{(x + \cos(x))}dx \rightarrow$ integrate $\rightarrow y(x) = \frac{1}{2}x^2 + \sin(x) + C$

Since $y(1) = 7$, we have the following:

$7 = \frac{1}{2}(1)^2 + \sin(x) + C \implies C = 7 - \frac{1}{2} - \sin(1) \implies C = \frac{13}{2} - \sin(1)$

Thus, we have the following solution:

Solution: $y(x) = \frac{1}{2}x^2 + \sin(x) + \frac{13}{2} - \sin(1)$


Solving Differential Equations using Definite Integrals

Take a look at the following definition: $\int^a_b f(x)dx = F(x)\Big|_a^b = F(b) - F(a)$

We can use this method to solve the exact problem in the last example:

Example: Finding the solution to a differential equation with initial conditions using definite integral method:

$y' = x + \cos(x)$ with $y(1) = 7$

We know that the $y(1) = 7$ implies that $x_o = 1$ and $y_o = 7$. Now let's rewrite the problem and integrate:

$\int^x_{x_o}\frac{dy}{ds}ds = \int^x_{x_o}s + \cos(s)ds$

Don't be intimitadated by the $s$'s. All I did was replace all of the $x$'s in the intial problem to $s$'s. This is because we don't want to confuse integration of $s$'s with the $x$ bounds. This method is commonly known as adding a "dummy variable". After integration we get:

$y(s)\Big|_{s=x_o}^{s=x} = (\frac{1}{2}s^2 + \sin(s))\Big|_{s=x_o}^{s=x} \implies y(x) - y(x_o) = (\frac{1}{2}x^2 + \sin(x)) - (\frac{1}{2}x_o^2 + \sin(x_o)) \implies$

$y(x) = \frac{1}{2}x^2 + \sin(x) - (\frac{1}{2}x_o^2 + \sin(x_o)) + y(x_o) \implies$

$y(x) = \frac{1}{2}x^2 + \sin(x) - \frac{1}{2}(1)^2 - \sin(1) + y(1) \implies$

$y(x) = \frac{1}{2}x^2 + \sin(x) - \frac{1}{2} - \sin(1) + 7 \implies$

Solution: $y(x) = \frac{1}{2}x^2 + \sin(x) + \frac{13}{2} - \sin(1)$


The solutions from using the definite integral method and the indefinite integral method are equivalent! This is nice and useful, sometimes one method is faster or esaier to understand than the other.

Special Integrals

When solving these differential equations, it gets difficult to integrate at times. Luckily, some very smart people have solved hard integral problems. If you'd like to understand them more they can be found in integration books or on the web, but here are a few of them that we might find useful:

a) $ln(x) = \int_1^x\frac{1}{s}ds$

b) $\arctan(x) = \int_0^x\frac{1}{1 + s^2}ds$

c) $erf(x) = \int_0^x\frac{2}{\sqrt(\pi)}e^{-s^2}ds$

d) $Si(x) = \int_1^x\frac{\sin(s)}{s}ds$

Example: Find solution of differential equation using one of our definitions from above

$\frac{dy}{dx} = e^{-x^2}$ with $y(0) = 0$

First integrate: $\int_{s=x_o}^{s=x}\frac{dy}{ds}ds = \int_{s=x_o}^{s=x}e^{-s^2}ds \implies y(s)\Big|_{s=x_o}^{s=x} = \int_{s=x_o}^{s=x}e^{-s^2}ds \implies y(x) - y(x_o) = \int_{s=x_o}^{s=x}e^{-s^2}ds \implies y(x) = \int_{0}^{x}e^{-s^2}ds + 0$

Now notice that we can multiply both sides by $(\frac{\sqrt{\pi}}{2} * \frac{2}{\sqrt{\pi}})$. Since the this is equivalent to 1, we can leave the left hand side alone, but we can change the right hand side to be the following:

$y(x) = \int_{0}^{x}e^{-s^2}ds = \frac{\sqrt{\pi}}{2} * \frac{2}{\sqrt{\pi}} * \int_{0}^{x}e^{-s^2}ds = \frac{\sqrt{\pi}}{2} * erf(x)$

Solution: $y(x) = \frac{\sqrt{\pi}}{2} * erf(x)$


Piecewise Defined Functions

A piecewise defined function, in simple words, is a function where in a certain region, the function is different than in another region. The following are examples of piecewise defined functions (let $\frac{dy}{dx} = f(x)$):

Example 1: \begin{equation} f(x) = \left\{ \begin{array}{ll} x^2 & \quad x \lt 2 \\ 1 & \quad x \geq 2 \end{array} \right. \end{equation}

Example 2: \begin{equation} |x| = \left\{ \begin{array}{ll} -x & \quad x \lt 0 \\ x & \quad x \geq 0 \end{array} \right. \end{equation}

Special Piecewise Defined Functions

A few a these "special" piecewise defined functions are the step function and the ramp function:

Step Function: \begin{equation} f(x) = \text{step}(x) = \left\{ \begin{array}{ll} 0 & \quad x \lt 0 \\ 1 & \quad x \gt 0 \end{array} \right. \end{equation}

Ramp Function: \begin{equation} f(x) = \text{ramp}(x) = \left\{ \begin{array}{ll} 0 & \quad x \lt 0 \\ x & \quad x \gt 0 \end{array} \right. \end{equation}

Let's now delve into solving Example 1, the Piecewise Defined Function.

Example: Find solution to the Piecewise Defined Function.

\begin{align} \frac{dy}{dx} = f(x) = \left\{ \begin{array}{ll} x^2 & \quad x \lt 2 & \text{(case 1)} \\ 1 & \quad x \geq 2 & \text{(case 2)} \end{array} \;\;\;\;\;\;\; \text{with} \;\; y(0) = 0 \right. \end{align}

We need to solve this problem case by case. Let's start with the first case:

Case 1: $x \lt 2$

$\int_0^x\frac{dy}{dx}ds = \int_0^xs^2ds \rightarrow$

$y(x) - y(0) = \frac{1}{3}s^3\Big|_0^x \rightarrow$

$y(x) = \frac{1}{3}x^3$

We have found that for the first case, the solution to the differential equation is $y(x) = \frac{1}{3}x^3$. Now, onto the second case.

Case 2: $x \geq 2$

$\int_0^x\frac{dy}{dx}ds = \int_0^xf(x)ds \rightarrow $

$y(s)\Big|_0^x = \int_0^2f(s)ds + \int_2^xf(s)ds \rightarrow $

$y(x) - y(0) = \int_0^2s^2ds + \int_2^x(1)ds \rightarrow $

$y(x) = \frac{1}{3}s^3\Big|_0^2 + s\Big|_2^x \rightarrow $

$y(x) = (\frac{8}{3} - 0) + (x - 2) \rightarrow $

$y(x) = x + \frac{8}{3} - 2 \rightarrow $

$y(x) = x + \frac{2}{3}$

We have now found the solution to the differential equation for the second case. Now, combining the case 1 and case 2, we can display our final solution as:

Solution: \begin{align} y(x) = \left\{ \begin{array}{ll} \frac{1}{3}x^3 & \quad x \lt 2 \\ x + \frac{2}{3} & \quad x \geq 2 \end{array} \right. \end{align}


Another example, a more complicated one involving the stair function.

Example: Find solution to the Piecewise Defined Function.

\begin{equation*} \text{Let} \;\;\; \frac{dy}{dx} = f(x) \;\;\;\;\;\;\; f(x) = \begin{cases} \text{stair}(x) & x \lt 4 \\ 0 & x \geq 4 \end{cases} \;\;\; \quad \text{stair}(x) = \begin{cases} 0 & x < 0 \\ 1 & 0 \leq x \lt 1 \\ 2 & 1 \leq x \lt 2 \\ 3 & 2 \leq x \lt 3 \\ \vdots & \vdots\\ \end{cases} \end{equation*}

We are going to solve this problem just like we did the last. We have to break it down into steps:

Case 1: $x \lt 0$

$\int_0^x\frac{dy}{dx}ds = \int_0^xf(s)ds \implies y(s)\Big|_0^x = \int_0^x0ds \implies \fbox{y(x) - y(0) = 0}$

Case 2: $0 \leq x \lt 1$

$\int_0^x\frac{dy}{dx}ds = \int_0^xf(s)ds \implies y(s) - y(0) = \int_0^x1ds \implies$

$y(x) - y(0) = s\Big|_0^x \implies y(x) - y(0) = x - 0 \implies \fbox{y(x) - y(0) = x}$

Case 3: $1 \leq x \lt 2$

$\int_0^x\frac{dy}{dx}ds = \int_0^xf(s)ds \implies \int_0^1\frac{dy}{dx}ds + \int_1^x\frac{dy}{dx}ds = \int_0^1f(s)ds + \int_1^xf(s)ds \implies$

$y(s)\Big|_0^1 + y(s)\Big|_1^x = \int_0^1ds + \int_1^x2ds \implies y(1) - y(0) + y(x) - y(1) = s\Big|_0^1 + 2s\Big|_1^x \implies$

Notice that in the above, $y(1) - y(1)$ cancels and we end up with:

$y(x) - y(0) = 1 - 0 + 2x - 2 \implies \fbox{y(x) - y(0) = 2x - 1}$

Case 4: $2 \leq x \lt 3$

$\int_0^x\frac{dy}{dx}ds = \int_0^xf(s)ds \implies y(s)\Big|_0^x = \int_0^1(1)ds + \int_1^2(2)ds + \int_2^x(3)ds \implies$

$y(x) - y(0) = (1) + 2s\Big|_1^2 + 3s\Big|_2^x \implies y(x) - y(0) = 1 + (4 - 2) + (3x + 6) \implies$

$y(x) - y(0) = 1 + 2 + 3x - 6 \implies \fbox{y(x) - y(0) = 3x - 3}$

Case 5: $3 \leq x \lt 4$

$\int_0^x\frac{dy}{dx}ds = \int_0^xf(s)ds \implies y(x) - y(0) = \int_0^1(1)ds + \int_1^2(2)ds + \int_2^x(3)ds + \int_3^x(4)ds \implies$

$y(x) - y(0) = 1 + 2 + 3s\Big|_2^3 + 4s\Big|_3^x \implies y(x) - y(0) = 3 + 9 - 6 + 4x - 12 \implies$

$\fbox{y(x) - y(0) = 4x - 6}$

Case 6: $x \leq 4$

$\int_0^x\frac{dy}{dx}ds = \int_0^xf(s)ds \implies y(x) - y(0) = \int_0^1(1)ds + \int_1^2(2)ds + \int_2^x(3)ds + \int_3^x(4)ds + \int_4^x(0)ds \implies$

$y(x) - y(0) = 1 + 2 + 3 + 4s\Big|_3^4 + 0 \implies y(x) - y(0) = 1 + 2 + 3 + 16 - 12 \implies \fbox{y(x) - y(0) = 10}$

We have solved for each case! Now we can put everything into a nice solution:

Solution: \begin{equation*} y(x) = y(0) + \begin{cases} 0 & x < 0 \\ x & 0 \leq x \lt 1 \\ 2x - 1 & 1 \leq x \lt 2 \\ 3x - 3 & 2 \leq x \lt 3 \\ 4x - 6 & 3 \leq x \lt 4 \\ 10 & 4 \leq x \end{cases} \end{equation*}


The Logistic Model: Population Density

Now we will be getting into more complicated differential equations where we will use a method called separation of variables. These types of function will have the following Stanard Form where the right hand side of the equation involves two variables instead of one (x and y for example, instead of just x):

\begin{equation*} \begin{cases} \frac{dy}{dx} = f(x, y) \\ y(x_o) = y_o \end{cases} \end{equation*}

A popular differential equation of this form is called the logistic equation, where $t$ is time and $P$ is population. $\alpha$ and $\beta$ are rates and $P_o$ is initial population:

\begin{equation*} \text{Logistic Equation:} \;\;\;\begin{cases} \frac{dP}{dt} = \alpha P - \beta P^2 \\ P(0) = P_o \end{cases} \end{equation*}

The goal is to solve for the population using separation of variables. Let's look at some examples:

Example: Find constant solution of differential equation.

$x^2\frac{dy}{dx} - 4x = 6$

The first step is to put the equation into standard form:

$x^2\frac{dy}{dx} - 4x = 6 \implies \frac{dy}{dx} = \frac{4x + 6}{x^2}$

Since there was no $y$ in the problem, we need not place it on the other side of the equation. So we can directly integrate and find the solution:

$\int\frac{dy}{dx}dx = \int\frac{4x + 6}{x^2}dx \implies y(x) = \int\frac{4}{x} + \frac{6}{x^2}dx \implies$

Solution: $y(x) = 4ln|x| - 6x^{-1} + C$


Some problems we cannot solve so easily. Some are more intuitive and we have to guess and check for a solution instead.

Example: Find the constant solution of differential equation.

$\frac{dy}{dx} + x^2y^2 = x^2$

The first step is to put the equation into standard form:

$\frac{dy}{dx} = x^2 - x^2y^2 \implies \frac{dy}{dx} = x^2(1 - y^2)$

What if we consider $y$ as a constant? If $y$ is a constant, then we know that the derivative of $y$ must be 0: $\frac{dy}{dx} = \frac{d}{dx}($constant$) = 0$

$\frac{dy}{dx} = 0 = x^2(1 - y^2) = x^2(1 - y)(1 + y)$

Solution: Let $y = 1$ or $-1$


Let's try another exmaple like this, but a little more complicated.

Example: Find the constant solution of differential equation.

$\frac{dy}{dx} = (y - 2x)(y^2 - 9)$

Let's find the constant solution, again, meaning that $\frac{dy}{dx} = 0$

$(y - 2x)(y^2 - 9) = 0 \implies (y - 2x)(y - 3)(y + 3) \implies y = 3, -3$, and $2x$

Since we are only looking for the constant solution, we end up with:

Solution: Let $y = 3$ or $-3$


Partial Differentiation

Theorem 3.2

Suppose we consider the IVP (initial value problem):

\begin{equation*} \begin{cases} \frac{dy}{dx} = F(x, y) \\ y(x_o) = y_o \end{cases} \end{equation*}

Assume that $F(x, y)$ and $\frac{\partial f}{\partial y}$ are boath continuous on $\mathbb{R}$ defined as:

$\mathbb{R} = \{(x, y) | \alpha < x < \beta, -\infty < y < \infty\}$, where $x_o \in (\alpha, \beta)$. Then the IVP has a unique solution on $x_o \in (\alpha, \beta)$. The solution is continuous and its derivative is also continuous.

For example of this theroem, take a look at the following:

\begin{equation*} \begin{cases} \frac{dy}{dx} = e^{xy} \\ y(1) = 3 \end{cases} \end{equation*}

$F(x, y) = e^{xy}$ is continuous everywhere. $\frac{\partial F}{\partial y} = \frac{\partial}{\partial x}e^{xy} = e^{xy} * \frac{\partial}{\partial y}(xy) = e^{xy} * x = xe^{xy}$. And this is continuous everywhere. Thus the IVP has a unique solution everywhere in the $xy$ plane.

Partial Derivatives

This is when we are taking the derivative of a function with respect to a specific variable. So far we've only derived with respect to $x$ but we can also derive with respect to $y$ or any other such variable:

$\frac{\partial F}{\partial y}(x, y) = \lim_{\Delta y\to\infty}\frac{F(x, y + \Delta y) - F(x, y)}{\Delta y}$

$\frac{\partial F}{\partial x}(x, y) = \lim_{\Delta x\to\infty}\frac{F(x + \Delta x, y) - F(x, y)}{\Delta x}$

Let's try a little example:

Example: What is the partial derivative with respect to $x$ and $y$ if $F(x, y) = \frac{x + y}{x - y}$

Solutions:

$\frac{\partial F}{\partial x} = \frac{(1)(x - y) - (1)(x + y)}{(x - y)^2} = \frac{x - y - x - y}{(x - y)^2} = \frac{-2y}{(x - y)^2}$

$\frac{\partial F}{\partial y} = \frac{(1)(x - y) - (-1)(x - y)}{(x - y)^2} = \frac{x - y + x + y}{(x - y)^2} = \frac{2x}{(x - y)^2}$


Another quick example of taking the derivative with respect to $y$:

Example: What is the partial derivative with respect to $y$ if $F(x, y) = x^2y^2 + xy\cos(xy)$

Solution:

$\frac{\partial F}{\partial y} = x^2(2y) + (x*1)\cos(xy) + (xy)(-\sin(xy) * x) = 2x^2y + x\cos(xy) - x^2y\sin(xy)$


Separable Equations

Earlier I mentioned separable equations but dismissed them and didn't provide any good examples. Now we will get into them. A separable equation is one that has the following form:

$\frac{dy}{dx} = F(x, y) = f(x) * g(y)$

Meaning, we can separate the equation into two functions, one function containing only $y$ and one containing only $x$.

Example: What is $g(y)$ and $f(x)$ in the following separable equation?: $\frac{dy}{dx} = x^2 - x^2y^2$

Separate the $x$'s and the $y$'s:

$\frac{dy}{dx} = x^2(1 - y^2)$

Solution: $f(x) = x^2$ and $g(y) = (1 - y^2)$


Now let's try solving the differential equation in the example above using separation of variables:

Example: Find the solution of the first order differential equation using separation of variables: $\frac{dy}{dx} = x^2(1 - y^2)$

Step 1: Multiply both sides by an infinitesimal in $x \rightarrow dx$

$\frac{dy}{dx} * dx = x^2(1 - y^2) * dx$

Step 2: From calculus, we know that $\frac{dy}{dx}dx = dy$

$dy = x^2(1 - y^2)dx$

Step 3: Move the dependent variable factors to the left hand side of the equation

$\frac{1}{1 - y^2}dy = x^2dx$

(Notice how the left hand side only deals with $y$ and the right hand side only deals with $x$)

Step 4: Integrate both sides with respect to the variables

$\int\frac{1}{1 - y^2}dy = \int x^2dx$

Let's first integrate the left hand side, to do this we need to use partial fractions:

$\frac{1}{(1 - y)(1 + y)} = \frac{A}{1 - y} + \frac{B}{1 + y} \implies 1 = A(1 + y) + B(1 - y)$

If we let $y = -1$, then $1 = B(1 + 1) + 0 \implies B = \frac{1}{2}$

If we let $y = 1$, then $1 = A(1 + 1) + 0 \implies A = \frac{1}{2}$

Now that we've solved for $A$ and $B$, we can rewrite our integration of the left hand side to be:

$\int\frac{\frac{1}{2}}{1 - y} + \frac{\frac{1}{2}}{(1 + y)}dy = -\frac{1}{2}ln|1 - y| + \frac{1}{2}ln|1 + y| + C$

Now after integrating the right hand side of the equation at the beginning of step 4, we can see it is $\frac{1}{3}x^3 + C$ and now let's combine the $C$'s from both integrations to get:

$-\frac{1}{2}ln|1 - y| + \frac{1}{2}ln|1 + y| = \frac{1}{3}x^3 + C \implies \frac{1}{2}ln|\frac{1 + y}{1 - y}| = \frac{1}{3}x^3 + C \implies ln|\frac{1 + y}{1 - y}| = \frac{2}{3}x^3 + 2C$

After multiplying a 2 everywhere, we see that we get $2C$, but $C$ is just a constant so $2C$ is a constant. So we can leave $2C$ to be $C$. Now let's exponentiate both sides:

$\frac{1 + y}{1 - y} = e^{\frac{2}{3}x^3 + C} \implies \frac{1 + y}{1 - y} = e^{\frac{2}{3}x^3} + e^{C} \implies \frac{1 + y}{1 - y} = Ae^{\frac{2}{3}x^3}$

Since $e^C$ is just another constant, we can just multiply it and denote it as a new constant $A$. Now let's simplify even further:

$(1 + y) = (1 - y)Ae^{\frac{2}{3}x^3} \implies y(1 + Ae^{\frac{2}{3}x^3}) = Ae^{\frac{2}{3}x^3} - 1 \implies $

Solution: $y(x) = \frac{Ae^{\frac{2}{3}x^3} - 1}{Ae^{\frac{2}{3}x^3} + 1}$


Another quick example:

Example: Find the solution of the differential equation: $\frac{dP}{dt} = \alpha P$

We can solve this equation by separation of variables. Note that $f(t) = \alpha$ and $g(P) = P$:

$\frac{dP}{dt}dt = \alpha Pdt \implies dP = \alpha Pdt \implies \frac{1}{P}dP = \alpha dt \implies \int\frac{1}{P}dP = \int\alpha dt \implies$

$ln|P| = \alpha t + C \implies e^{ln|P|} = e^{\alpha t + C} \implies P(t) = e^{\alpha t} * e^{C} \implies$

Solution: $P(t) = Ae^{\alpha t}$


Another problem of the same form but with initial conditions:

Example: Find the solution of the differential equation: $\frac{dy}{dx} = \frac{y}{x}$ with $y(1) = 2$

First off, let's recall that $y(1) = 2$ indicates that $y_o = 2$ and $x_o = 1$. Now let's separate varibales:

$\frac{dy}{dx}dx = \frac{y}{x}dx \implies dy = \frac{y}{x}dx \implies \frac{1}{y}dy = \frac{1}{x}dx \implies \int_{y_o}^y\frac{1}{s}ds = \int_{x_o}^x\frac{1}{c}dc \implies$

$ln|s|\Big|_{y_o}^y = ln|c|\Big|_{x_o}^x \implies ln|y| - ln|y_o| = ln|x| - ln|x_o| \implies ln|y| = ln|x| + ln|2| \implies e^{ln|y|} = e^{ln|2x|} \implies $

Solution: $y(x) = 2x$


Earlier we discussed finding the constant solutions to a differential equation. Let's see this in action in another equation where we also solve the differential equation using separation of variables:

Example: Find the solution of the differential equation: $\frac{dy}{dx} = 2x(y - 5)$

First let's find the constant solutions. Remember that if $\frac{dy}{dx} = 0$, then the solution $y$ must be a constant.

$\frac{dy}{dx} = 0 \implies 0 = 2x(y - 5) \implies y - 5 = 0 \implies y = 5$

We have found the constant solution, now let's solve for the general solution of the differential equation:

$\frac{dy}{dx}dx = 2x(y - 5)dx \implies dy = 2x(y - 5)dx \implies \frac{1}{y - 5}dy = 2xdx \implies $

$\int\frac{1}{y - 5}dy = \int2xdx \implies ln|y - 5| = x^2 + C \implies y - 5 = e^{x^2 + C} \implies y = Ae^{x^2 + C}$

Recall that when we separate what's in the exponent, we create a new constant and our general solution becomes: $y(x) = Ae^{x^2} + 5$

Now, notice that since our constant solution is 5, we can plug in any number $x$ into the general solution and we'll still find that the exact solution is $y = 5$. For example, let's say $x = 12.9$. Then:

$y(12.9) = Ae^{12.9^2} + 5 \implies 5 = Ae^{12.9^2} + 5 \implies 0 = Ae^{12.9^2} \implies A = 0$.

If $A = 0$, then our solution is: $y(x) = 0e^{x^2} + 5 \implies $

Solution: $y(x) = 5$


In many examples, it is smart to first look for a constant solution and then if there isn't one, we have to try other methods to find the solution. In the example above, we first searched for a constant solution and found one! In some cases, we may not be so lucky. For example, let's look at the following:

Example: Find the solution of the differential equation: $\frac{dy}{dx} - x^2y^2 = x^2$

First look for constant solutions!

$\frac{dy}{dx} = 0 = F(x, y) = f(x) * g(y)$

We need to ensure that $g(y) = 0$. If $g(y) = 1 + y^2$, then we need $(1 + y^2) = 0$. For this to be 0, we need $y^2 + 1 = 0 \implies y^2 = -1 \implies y = \sqrt{-1} = i$.

Since $y = i$, we know that there can be no constant solution! Now let's just try to solve for the general solution using separation of variables:

$dy = x^2(1 + y^2)dx \implies \frac{1}{1 + y^2}dy = x^2dx \implies \int\frac{1}{1 + y^2}dy = \int x^2dx \implies \arctan(y) = \frac{1}{3}x^2 + C \implies$

Solution: $y(x) = \tan(\frac{1}{3}x^3 + C)$


First Order Linear ODE's

Some equations we cannot simply separate variables and integrate. These such equations have the following form:

$\frac{dy}{dx} + p(x)y = f(x)$

It is impossible to separate the above expression into $x$'s on one side and $y$'s on the other. Therefore, we need a new method to solve this differential equation.

The method we will use is based on the product rule of differentiation:

$\frac{d}{dx}(\mu y) = \frac{d\mu}{dx}y + \mu \frac{dy}{dx}$

Compare this to the standard form of the 1st order linear ODE: $\frac{dy}{dx} + p(x)y = f(x)$

Multiply this ODE by $\mu(x):$

$\mu(x)\frac{dy}{dx} + \mu(x)p(x)y = \mu(x)f(x)$.

The product rule indicates:

$\mu\frac{dy}{dx} + \frac{d\mu}{dx}y \implies \frac{d\mu}{dx} = \mu(x)p(x) \implies \frac{d\mu}{dx} = \mu p(x) \implies \frac{d\mu}{dx}dx = \mu pdx \implies d\mu = \mu pdx \implies \frac{1}{\mu}d\mu = pdx$

Now we integrate:

$\int\frac{1}{\mu}d\mu = \int pdx \implies \mu = e^{\int p(x)dx} \rightarrow $ integrating factor $\rightarrow \frac{d}{dx}(\mu p) = \frac{d\mu}{dx}y + \mu\frac{dy}{dx}$

To clear this up, let's define some steps when we come across a problem with the form $\frac{dy}{dx} + p(x)y = f(x)$:

Steps on solving differential equation with the form: $\frac{dy}{dx} + p(x)y = f(x)$:

Step 1: Compute an integrating factor:

$\mu = e^{\int p(x)dx}$

Step 2: Write out new equation to solve as:

$\frac{d}{dx}(\mu y) = \mu f(x)$

Step 3: Integrate.

$\mu y = \int \mu(s)f(s)ds + C \implies y = \frac{1}{\mu(s)} \int\mu(s)f(s)ds + \frac{C}{\mu(x)}$


Finally, let's see this process in action with a few examples:

Example: Find the solution of the differential equation: $y' + \frac{4}{x}y = x^3$

Firstly, let's identify that $f(x) = x^3$ and $p(x) = \frac{4}{x}$.

If $p(x) = \frac{4}{x}$, then we have:

$\mu(x) = e^{\int p(x)dx} = e^{\int\frac{4}{x}dx} = e^{4ln|x|} = e^{ln|x^4|} = x^4$

Now we solve $\frac{d}{dx}(\mu y) = \mu f(x)$:

$\frac{d}{dx}(\mu y) = \mu f(x) \implies \frac{d}{dx}(x^4y) = x^3x^4 \implies \frac{d}{dx}(x^4y) = x^7$

Now we need to integrate:

$\int\frac{d}{dx}(x^4y)dx = \int x^7dx \implies x^4y = \frac{1}{8}x^8 + C \implies$

Solution: $y(x) = \frac{1}{8}x^4 + Cx^{-4}$


Example: Find the solution of the differential equation: $y' + 6xy = x^2$

First find the integrating factor:

$\mu = e^{\int 6xdx} = e^{3x^2}$

$\frac{d}{dx}(\mu y) = \mu x^2 \implies \frac{d}{dx}(e^{3x^2} y) = e^{3x^2}x^2 \implies \int\frac{d}{dx}(e^{3x^2}y)dx = \int e^{3x^2}x^2dx \implies e^{3x^2}y = \int e^{3x^2}x^2dx + C \implies$

Solution: $y(x) = e^{-3x^2}\int e^{3x^2}x^2dx + Ce^{-3x^2}$


When coming across differential equations like this, the key is to first try to rewrite the equation into the form: $\frac{dy}{dx} + p(x)y = f(x)$. For example, if we have the following function: $e^{2x}\frac{dy}{dx} = 40 - 3e^xy$. Then we rewrite it as: $\frac{dy}{dx} + 3e^{-x}y = 40e^{-2x}$

U Substitution

What if we come accross an equation we haven't learned to solve before? Say we have a differential equation that we cannot solve by direct integration, separation of variables, and it is not 1st Order Linear. The trick to solving such an equation is by u substitution. Let's look at an example:

Example: Find the solution of the differential equation: $\frac{dy}{dx} = (x + y)^2$

The equation cannot be solved by direct integration, we cannot separate the $x$'s and $y$'s, and it is not first order linear. Let's try a $u$ substitution by letting $u = x + y$:

$u = x + y \implies y = u - x \implies \frac{dy}{dx} = \frac{du}{dx} - 1$

Plug in:

$\frac{du}{dx} - 1 = (u)^2 \implies \frac{du}{dx} = u^2 + 1$

Now this is separable and we can solve by separation of variables (separating $u$ and $x$):

$\frac{du}{dx}dx = (1 + u^2)dx \implies du = (1 + u^2)dx \implies \frac{1}{1 + u^2}du = dx \implies$

$\int \frac{1}{1 + u^2}du = \int 1dx \implies \arctan(u) = x + C \implies u = \tan(x + C) \implies$

Now we plug in $u = x + y$

$x + y = \tan(x + C) \implies$

Solution: $y(x) = \tan(x + C) - x$
